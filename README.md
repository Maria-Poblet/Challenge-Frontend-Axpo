Contract Onboarding - Blazor App -- María Poblet --ENG
Interactive web application for contract visualization and management ("Contract Onboarding"). Developed with .NET 8 and Blazor Interactive Server, following a modern and scalable component architecture.

Run the Project
Prerequisites: Ensure you have the .NET 8 SDK installed.

Open a terminal in the project's root folder.

Architecture and Structure
The project follows a "Feature-First" architecture and clean design principles:

/Components/UI: Reusable visual components agnostic to the business (e.g., Accordion).

/Components/Features: Specific business components (e.g., ContractDetails).

/Services: Business logic and simulated data retrieval (ContractService).

/Models: Strongly typed data definitions.

Styles: Use of CSS Isolation (.razor.css) for modular styles and global CSS variables for the design system.

Highlighted Features
Adaptive Design: Fully responsive interface (Mobile, Tablet, Desktop).

Smart Components: The accordion manages its own open/close state using CSS Grid animations (grid-template-rows).

Developed with Blazor

Contract Onboarding - Blazor App -- María Poblet --ESP

Aplicación web interactiva para la visualización y gestión de contratos ("Contract Onboarding"). Desarrollada con .NET 8 y Blazor Interactive Server, siguiendo una arquitectura de componentes moderna y escalable.

Ejecutar el Proyecto

Requisitos: Asegúrate de tener instalado el .NET 8 SDK.

Abre una terminal en la carpeta raíz del proyecto.


Arquitectura y Estructura

El proyecto sigue una arquitectura "Feature-First" y principios de diseño limpio:

/Components/UI: Componentes visuales reutilizables y agnósticos al negocio (ej. Accordion).

/Components/Features: Componentes de negocio específicos (ej. ContractDetails).

/Services: Lógica de negocio y obtención de datos simulada (ContractService).

/Models: Definiciones de tipos de datos fuertemente tipados.

Estilos: Uso de CSS Isolation (.razor.css) para estilos modulares y variables CSS globales para el sistema de diseño.


Características Destacadas

Diseño Adaptativo: Interfaz totalmente responsive (Mobile, Tablet, Desktop).

Componentes Inteligentes: El acordeón gestiona su propio estado de apertura/cierre con animaciones CSS Grid (grid-template-rows).
